WP2Laconica generates updates for laconica.

== Description ==

WP2Laconica generates updates for laconica (you can add custom services based on Laconica ) when a new post is published.
Plugin based on WordIdentica and was modified for personal use :)



== Installation ==

Requirements:
- WordPress 2.6
- php-curl


1. Upload `wp2laconica` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure your login, password and laconica based service.
